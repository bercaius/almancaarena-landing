# 🇩🇪🐶 Bello Almanca Arena

Bello Almanca Arena, Almanca öğrenmeyi daha eğlenceli, etkili ve motive edici hale getirmek için geliştirilen modern bir mobil öğrenme platformudur.

Uygulama; oyunlaştırma, başarı sistemleri, günlük görevler, widget desteği ve yapay zeka destekli koçluk özelliklerini bir araya getirerek kullanıcıların Almanca öğrenme sürecini daha sürdürülebilir hale getirmeyi amaçlar.

---

## ✨ Temel Özellikler

### 🐶 Bello Maskotu

Bello, uygulamanın dijital öğrenme arkadaşıdır.

* Başarıları kutlar
* Motivasyon sağlar
* Günlük görevlerde kullanıcıya eşlik eder
* AI Koç sistemiyle entegre çalışır

---

### 🎮 Oyunlaştırılmış Öğrenme

* XP sistemi
* Başarımlar (Achievements)
* Günlük görevler
* Seri koruma sistemi
* Seviye ilerleme mantığı
* Motivasyon widgetları

---

### 🧠 AI Coach

Kullanıcılar kendi Google AI Studio API anahtarlarını kullanarak Bello AI Coach'a erişebilirler.

AI Coach:

* Almanca sorularını cevaplayabilir
* Kelime açıklamaları yapabilir
* Dilbilgisi desteği sağlayabilir
* Öğrenme tavsiyeleri verebilir

---

### 📚 Almanca İçerikler

* Kelime çalışmaları
* Konu bazlı üniteler
* Testler
* Mini oyunlar
* Günlük pratik araçları

---

### 📱 Widget Desteği

Ana ekrana eklenebilen Bello widgetları:

* Motivasyon mesajları
* Seri koruma hatırlatmaları
* Günlük öğrenme teşvikleri
* Başarım bildirimleri

---

## 🔒 Gizlilik Odaklı Tasarım

Bello Almanca Arena mümkün olduğunca yerel çalışan bir mimari ile geliştirilmiştir.

* Kullanıcı verileri cihaz üzerinde saklanır
* Merkezi kullanıcı veritabanı kullanılmaz
* API anahtarları geliştirici sunucularında depolanmaz
* Kullanıcı verileri minimum seviyede tutulur
